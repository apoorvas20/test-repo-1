## Falconer

Falconer is an AI-powered knowledge management and documentation platform. It helps teams create, collaborate on, and discover organizational knowledge in one place.

### What it does

Falconer pairs intelligent document creation with fast search and discovery. Write docs, publish them to your org, and find any answer across your team's knowledge.

### Key features

**Document creation**

- Write from scratch or generate content with AI prompts
- Highlight text to expand, clarify, or restructure it inline
- Full markdown with syntax-highlighted code and copy buttons
- Tables, lists, and code blocks out of the box

**Falcon AI**

- Falcon is available from the home page, editor sidebar, Slack, and inline text selection
- Same core skills everywhere: search, Q&A, writing docs, filing Linear issues
- The editor sidebar can edit the current document directly
- Selecting text opens an inline prompt to rewrite, expand, simplify, or restructure

**Organization and publishing**

- Draft privately, then publish to company collections
- Use Organize mode to auto-restructure docs into a clean hierarchy
- Review proposed moves, new folders, and removals before applying
- Export any doc as markdown

**Search and discovery**

- Find information across all your team's documentation
- Explore related docs and knowledge connections
- Review and restore previous versions
- With a connected repo, Falcon can explore real git history: commits, blame, and diffs

**Collaboration**

- Add contextual comments for team discussion
- Move docs from private drafts to company-wide visibility
- Use @ to cross-reference other Falconer docs

### Tips

- Start in private collections to develop content before publishing
- Use @ references to build a connected knowledge network
- Use `/` for quick formatting commands
- Use arrow keys in the move modal to reorganize fast
